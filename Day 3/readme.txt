# Rollercoaster Ticket Calculator

Welcome to the Rollercoaster Ticket Calculator! This Python script helps you determine if you can ride a rollercoaster based on your height, calculates the ticket price based on your age, and optionally adds a charge for a photo.

5.Explanation:
  a)Welcome Message: Displays a welcome message.
  b)Height Input: Prompts the user to input their height.
  c)Bill Initialization: Initializes the bill amount.
  d)Height Check: Checks if the user meets the minimum height requirement.
  e)Age Input and Ticket Pricing: Prompts the user for their age and determines the ticket price based on age categories.
  f)Photo Option: Asks if the user wants a photo and adds the charge if they do.
  g)Final Bill: Displays the final bill amount.
  h)Height Requirement Not Met: Informs the user if they do not meet the height requirement.
